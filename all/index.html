DriveSiemens 6RA80 DC Drive (200 kW)
HMISiemens KTP900 Basic (WinCC Basic)
SensorsProximity switches, encoder, thermal sensors
ProtocolPROFINET IO
IndustrySteel Manufacturing (SAIL)

Project Overview

Complete automation retrofit for a 15-ton vertical lathe machine used in heavy disc and ring machining at Bokaro Steel. Replaced aging relay-based controls with a modern PLC/drive system.

Working Principle

The S7-1200 PLC manages the entire sequence: workpiece clamping → spindle speed ramp-up via 6RA80 DC drive → auto-feed engagement → cutting cycle → tool retract → unclamp. The DC drive enables smooth torque control across the full speed range required for heavy turning operations.

My Responsibilities

  • Full PLC program development (LAD + SCL)
  • 6RA80 DC drive parameterization and speed loop tuning
  • HMI screen design with real-time spindle RPM display
  • Safety interlock programming per IEC 62061
  • Site commissioning and operator training

Challenges & Solutions

  • Speed oscillation at low RPM → retuned current controller Kp/Ti
  • Encoder noise causing position faults → added hardware filter + PLC debounce
  • Legacy DC motor armature resistance mismatch → performed field weakening recalibration
10% cycle time reduction
Zero safety incidents post-commissioning
Auto mode uptime: 97.8%
